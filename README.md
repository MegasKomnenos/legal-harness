# legal-case-harness

Claude Code 기반 한국 행정법 문서 생성 프레임워크.

국민신문고 민원, 정보공개청구, 행정심판 보충서면 등 행정법 문서를 체계적으로 생성하기 위한 참조 체계(harness)와 Claude Code 스킬을 제공한다. 경북대학교 대상 3건의 실제 행정심판 사건에서 추출한 34개 법리, 26건 판례 인용, 6종 문서 템플릿을 포함한다.

## 핵심 구성요소

### `.claude/skills/legal-harness/` -- 법률 문서 생성 스킬

Claude Code에서 `/legal-harness [사건명] [문서유형]` 명령으로 법률 문서를 생성한다.

- 9단계 생성 파이프라인: 참조 파일 읽기 -> 사건 문서 통독 -> 판례 전문 통독 -> 사안 파악 -> 쟁점 정리 -> 문서 생성 -> 반복 자기 검증 -> PDF 출력
- 전문 전체 적재 원칙: 모든 판례와 법령을 snippet이 아닌 전문으로 읽어 컨텍스트에 직접 적재
- 반복 품질 검증: 14항목 체크리스트를 전 항목 충족 시까지 무한 반복

### `harness/` -- 법률 문서 생성 참조 체계

| 파일 | 내용 |
|------|------|
| `01_법리_데이터베이스.md` | 34개 법리 (ID, 판례, 판시사항 원문, 논증 구조, 적용 사례) |
| `02_판례_인용_사전.md` | 26건 판례 인용 형식 및 판시사항 원문 |

| `07_논증_가이드라인.md` | 논증상 오류 예시 및 교정 방법 |

| `09_문서_지도.md` | 디렉토리 전체 구조 및 사건별 문서 분류 |
| `10_유지보수_가이드.md` | 하네스 수정 시 정합성 유지 절차 |

### `.claude/skills/extract-documents/` -- 문서 추출 스킬

PDF, HWP, DOCX, 이미지, 오디오, 영상 등 모든 문서 형식에서 텍스트를 추출한다.

- 7가지 추출 방법: 직접 텍스트 읽기, PDF 텍스트 레이어, OCR, HWP 변환, DOCX XML 파싱, STT, 영상 프레임 OCR
- PDF 페이지별 판정 로직으로 텍스트/이미지/벡터 페이지를 자동 분류

## 사건 개요

| 사건 | 유형 | 피청구기관 | 상태 |
|------|------|-----------|------|
| 공사소음 | 공사 소음/비산먼지 정보공개 | 경북대학교 시설과 | 행정심판 진행 중 |
| 대동제 | 행사 용역 정보공개 | 경북대학교 학생지원과 | 행정심판 진행 중 |
| 성희롱 | 단톡방 성희롱 정보공개 | 경북대학교 인권센터 | 행정심판 진행 중 |

모든 사건 문서는 개인정보가 제거된 익명화 버전이다.

## 법리 범위

| 카테고리 | 법리 수 | 예시 |
|----------|---------|------|
| 정보공개법 | 22개 | 부분공개 원칙, 비공개 사유(제1~7호), 이유제시, 부존재, 전자적 공개, 입증책임 등 |
| 행정심판 | 4개 | 처분성, 부작위, 보완요청 법적 성격, 병합 심리 |
| 민원/징계 | 5개 | 형사-행정 분리, 기속적 의무, 피해자 보호, 학칙 개정 취지 |
| 환경법 | 3개 | 비산먼지 신고, 특정공사 사전신고, 발주처 책임 |

## 사용법

### Claude Code에서 사용

```
/legal-harness 공사소음 보충서면
/legal-harness 대동제 청구이유서
/legal-harness 성희롱 정보공개청구 별지
```

### 수동 참조

1. `03_문서_구조_템플릿.md`에서 문서 유형별 구조 확인
2. `01_법리_데이터베이스.md`에서 적용할 법리 선택
3. `02_판례_인용_사전.md`의 원문으로만 판례 인용
4. `04_표현_사전.md`의 정형 표현 사용
5. `05_품질_기준_및_검증.md` 체크리스트로 검증

## 문서 추출 도구

`extract_all.py`는 지정된 파일에서 텍스트를 추출하여 stdout에 출력한다.

```bash
python3 extract_all.py "법령_판례/판례/대법원_2003두8050.pdf"
python3 extract_all.py "법령_판례/경북대_내규/경북대학교_학칙(규정_제2856호).pdf"
```
