## 사용법

### Claude Code 스킬로 사용
```
/legal-harness 이 사안에 대해 정보공개청구서를 작성해줘
```

### 수동 참조
1. 작성할 문서 유형에 맞는 `03_문서_구조_템플릿.md`의 구조 확인
2. 사안에 적용할 법리를 `01_법리_데이터베이스.md`에서 선택
3. 판례 인용은 `02_판례_인용_사전.md`의 원문만 사용
4. 표현은 `04_표현_사전.md` 참조
5. 완성 후 `05_품질_기준_및_검증.md` 체크리스트로 검증
