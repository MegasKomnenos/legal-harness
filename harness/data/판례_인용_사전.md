# 판례 인용 사전

본 문서는 하네스에서 사용 가능한 모든 판례의 정확한 인용 형식과 판시사항 원문을 수록한다.
**생성 규칙**: 문서 생성 시 본 사전에 수록된 판례만 인용 가능. 수록되지 않은 판례 인용 시 별도 검증 필수.
