# 표현 사전

법률 문서에서 반복적으로 사용되는 정형 표현·어구·패턴의 사전.

---

## 1. 약칭 도입

| 패턴 | 예시 |
|------|------|
| 「법률명」(이하 '약칭'이라 합니다) | 「공공기관의 정보공개에 관한 법률」(이하 '정보공개법'이라 합니다) |
| (이하 '약칭'이라 합니다) | (이하 '본 사안'이라 합니다), (이하 '본 용역'이라 합니다) |
| 같은 법 | 정보공개법 → "같은 법 제14조" |
| 같은 조, 같은 항, 같은 호 | "같은 법 제9조 제1항 제5호 단서", "같은 호 단서 라목" |
| 동 규정 | "동 규정 제14조 제6항" |
| 위 판결 | "위 2009두19021 판결" |

---

## 2. 판례 인용

| 패턴 | 예시 | 사용 조건 |
|------|------|----------|
| 정식 인용 | 대법원 2011. 11. 24. 선고 2009두19021 판결 | 첫 인용 시 |
| 직접 인용 (판시 원문) | 대법원은 "~"라고 판시하였습니다 | 판시사항 원문을 그대로 옮길 때. 반드시 인용부호 `"` 사용 |
| 간접 인용 (요약) | 대법원은 ~는 취지로 판시하였습니다 | 판시를 요약·의역할 때. 인용부호 미사용 |
| ~고 판시한 바 있습니다 | 대법원은 ~고 판시한 바 있습니다 | 간접 인용 변형 |
| ~고 판단하였습니다 | 같은 판결은 ~고 판단하였습니다 | 간접 인용 변형 |
| ~고 보았습니다 | 위 판결은 ~고 보았습니다 | 간접 인용 변형 |
| 괄호 부기 참조 | (대법원 YYYY. M. D. 선고 XXXX두YYYY 판결 참조) | 문장 끝에 근거 표시 |
| 약식 후속 | "위 XXXX두YYYY 판결의 법리에 따르면" | 이미 정식 인용한 판례 재참조 |
| 복수 병렬 | (대법원 YYYY 판결, 대법원 YYYY 판결 등 참조) | 복수 판례 근거 |
| 재결 인용 | 국민권익위원회 YYYY. M. D. 재결 제YYYY-XXXXX호 | 재결"례" 아닌 재결 + 호수 |

## 3. 청구 목적·성격 선언

| 패턴 | 예시 |
|------|------|
| 목적 선언 | "본 청구의 목적은 ~ 확인하는 데 있습니다" |
| 비목적 선언 | "본 청구는 ~ 확인을 목적으로 하지 아니합니다" |
| 공익적 성격 | "본 청구는 헌법상 도출되는 국민의 알 권리 보장과 ~ 공익적 목적에 근거합니다" |
| 전제 선언 | "본 민원은 ~ 전제로, ~ 요청하는 데에 그 취지가 있습니다" |
| 한계 선언 | "청구인은 피청구기관이 현재 보유·관리하지 아니하는 정보를 새로 작성하거나 새로 취득하여 공개하여야 한다고 주장하는 것이 아닙니다" |

---

## 4. 제외 선언

| 패턴 | 예시 |
|------|------|
| 항목별 제외 | "본 항에 대하여 명시적으로 제외하는 부분:" |
| 공통 제외 | "본 청구는 청구 대상 전반에 걸쳐 다음 정보의 공개를 요구하지 아니하며, 해당 부분에 대한 정보공개법 제14조에 따른 비공개 처리(마스킹·삭제 등)를 수용합니다" |
| 개별 제외 | "피해자·진정인·조사대상자·조사위원·내부 관계자 등의 성명·주민등록번호·소속·연락처 등 개인식별정보" |

## 5. 부분공개 수용

| 패턴 |
|------|
| "정보공개법 제14조에 따른 마스킹·삭제 등의 방식에 의한 부분공개를 수용합니다" |
| "비공개 사유가 문제될 수 있는 부분은 정보공개법 제14조에 따라 마스킹·삭제 등의 방식으로 비공개 처리할 수 있습니다" |
| "비공개 처리 또는 비식별화 방식의 부분공개를 수용할 의사가 있습니다" |

## 6. 선제적 방어

| 패턴 | 예시 |
|------|------|
| 인식 표명 | "피청구기관이 본 청구에 대하여 정보공개법 제9조 제1항 제○호를 원용할 가능성이 있음을 청구인은 인식하고 있습니다" |
| 반박 전환 | "그러나 ~" / "다만 ~" |
| 비공개 시 요구 | "피청구기관이 같은 호를 원용하여 비공개 결정을 하려는 경우에는, 다음 사항을 문서별·항목별로 밝혀 주시기 바랍니다" |

## 7. 비공개 사유 원용 반박

| 패턴 |
|------|
| "단순히 '내부검토 자료' 또는 '의사결정 과정 중인 사항'이라는 추상적 사유만을 제시할 것이 아니라" |
| "추상적 우려만으로는 위 요건을 인정하기 어렵습니다" |
| "비공개로 보호되는 이익과 공개로 보호되는 이익을 비교·교량하여야 합니다" |
| "조사 공정성, 당사자 보호, 2차 피해 방지라는 추상적 명분만으로는 입증책임을 충족하기 어렵습니다" |

## 8. 위법 결론

| 패턴 |

## 10. 부존재 시 안내 요청

## 13. 톤·레지스터 규칙

| 규칙 | 설명 |
|------|------|
| 경어체 사용 | 모든 법률 문서는 경어체. "~합니다", "~바랍니다" |
| 정중하되 단호 | 감정적 표현 배제. 법리와 사실에 근거한 논증 |
| 주어 명시 | "피청구인은", "청구인은" — 주어를 명확히 |
| 능동태 우선 | "피청구인은 ~ 하였습니다" (수동태 최소화) |
| "~이라 합니다" 패턴 | "~이라 합니다" (번역투 피함) |
| 단정 회피 (민원) | "~에 해당한다고 사료됩니다", "~로 판단됩니다" |
| 단정 사용 (심판) | "~에 해당합니다", "~는 위법합니다" |

## 14. 문서 머리·말미 형식 규칙

> 본 절은 텍스트 수준의 형식을 규정한다. PDF 서식(폰트·여백·행간 등)은 `quality/PDF_서식.md`를 따른다.

| 항목 | 규칙 | 예시 |
|------|------|------|
| 사건 표시 | 콜론 없이 공백 2칸. 사건명 병기 | `사 건  {{사건번호}} {{사건명}}` |
| 당사자 표시 | 콜론 없이 공백 2칸. 성명 자간 벌림 | `청 구 인  {{청구인_성명_자간벌림}}` |
| 피청구인 | 기관명과 직위 분리 | `피청구인  {{피청구인}}` |
| 말미 서명 | (인)만 표기. 성명 자간 벌림 | `청구인  {{청구인_성명_자간벌림}}  (인)` |
| 수신처 | 공백 삽입 | `중앙행정심판위원회  귀중` |
| 날짜 | YYYY. M. D. (마침표+공백) | `2026. 6. 17.` |

## 15. 비표준 용어 교정 목록

| 비표준 (사용 금지) | 표준 (사용) | 비고 |
|-------------------|-----------|------|
| 가림 처리 | 비공개 처리, 마스킹 | "가림"은 일상어 |
| 범주화 | 분류별 공개, 범위별 공개 | 법령 용어 아님 |
| 구간화 | 금액 구간 표시, 범위별 공개 | 법령 용어 아님 |
| 재결례 | 재결 제○○호 | "례"가 아닌 호수로 특정 |
| 1), 가) | (1), (가) | 양쪽 괄호 필수 |

## 16. 지양 표현·패턴

| 지양 표현/패턴 | 대체 | 이유 |
|---------------|------|------|
| — (em dash) 소제목·본문 연결 | 괄호 또는 제목 분리 | 법률 문서 관행과 불일치 |
| "전혀 다릅니다" | "같다고 볼 수 없습니다" | 과장 표현 |
| "불가능합니다" | "있다고 보기 어렵습니다" | 절대적 단정 |
| "그 이유는 다음과 같습니다" | (삭제 후 바로 논증 시작) | AI 나열 도입부 |
| "~을 제시하지 못하였습니다" (단독 결론) | 왜 충족되지 않는지 직접 논증 | 소극적 논증 |
| "~을 정확히 읽지 않은 것으로 보입니다" | (삭제) | 상대방 능력 비하 |
| "대체로", "대략", "어느 정도", "일종의" | (삭제하고 단정적으로 서술) | 추정적·유보적 표현. 상대방 주장을 정확히 파악하지 못한 인상 |
