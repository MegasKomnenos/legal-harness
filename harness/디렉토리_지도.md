# 디렉토리 문서 지도

프로젝트 디렉토리 전체의 구조와 각 문서의 분류를 정리한다.
문서 생성 시 관련 사건 폴더의 문서를 빠짐없이 참조하기 위한 용도이다.

---

## 1. 최상위 디렉토리 구조

```
legal-case-harness/
├── cases/
│   ├── 01_공사소음/   <- 사건 1: 경북대 공사 소음·비산먼지 정보공개 사건
│   ├── 02_대동제/     <- 사건 2: 대동제 행사 용역 정보공개 사건
│   └── 03_성희롱/     <- 사건 3: 컴퓨터학부 단톡방 성희롱 사건
├── 법령_판례/         <- 공통 법령·판례·내규 자료
├── harness/           <- 하네스 참조 파일
│   ├── data/          <- 법리·판례·조문 데이터
│   ├── style/         <- 문서 서식·표현·논증
│   ├── quality/       <- 검증·PDF 사양
│   ├── scripts/       <- 검증·출력 스크립트
│   ├── 관리_절차.md   <- 판례 추가, case 추가/수정 절차
│   └── 디렉토리_지도.md <- 본 파일
├── .claude/           <- Claude Code 설정 및 스킬
└── extract_all.py     <- 문서 추출 스크립트
```

> **바이너리 문서 읽기**: PDF·HWP·DOCX 등을 직접 읽을 수 없을 때는 `python3 extract_all.py <파일경로>`를 Bash로 실행하여 추출된 텍스트를 stdout에서 읽는다.

## 4. 사건별 관련 문서 참조 가이드

문서 생성 시 해당 사건 폴더 내 문서뿐 아니라 아래 교차 참조가 필요하다.

### 공사 사건 문서 작성 시 참조 범위

1. `공사/` 폴더 전체
2. `법/` 중: 정보공개법·시행령, 공공기록물법·시행령, 행정심판법, 행정절차법, 소음진동관리법·시행규칙, 건설산업기본법·시행령, 건설기술진흥법, 건축법, 대기환경보전법·시행령·시행규칙, 관련 별표
3. `법/` 중 판례: 정보공개 관련 대법원 판례 전체
4. `소음/` 폴더 (관련 민원)

### 대동제 사건 문서 작성 시 참조 범위

1. `대동제/` 폴더 전체
2. `법/` 중: 정보공개법·시행령, 공공기록물법·시행령, 행정심판법, 행정절차법, 국가계약법, 국립대학회계법, 부정경쟁방지법
3. `법/` 중 판례: 정보공개 관련 대법원 판례 전체
4. `법/` 중 내규: 총학생회칙, 학생활동규정
5. `법/` 중 기타: 국립대학 예산집행 기본지침

### 성희롱 사건 문서 작성 시 참조 범위

1. `성희롱/` 폴더 전체
2. `기계과/` 폴더 (관련 증거 스크린샷)
3. `법/` 중: 정보공개법·시행령, 행정심판법, 행정절차법
4. `법/` 중 내규: 학칙, 상벌규정, 학생활동규정
5. `법/` 중 판례: 정보공개 관련 대법원 판례, 성희롱·징계 관련 판례

## 6. 바이너리 문서 추출

PDF, HWP, DOCX 등 바이너리 문서의 텍스트가 필요할 때 `extract_all.py`를 사용한다.

```bash
python3 extract_all.py <파일경로1> [파일경로2] ...
```

- 추출된 텍스트는 stdout에 출력된다.
- `.txt`, `.md` 파일은 추출 없이 Read 도구로 직접 읽는다.
- 판례 PDF 예시: `python3 extract_all.py "법령_판례/판례/대법원_2003두8050.pdf"`
