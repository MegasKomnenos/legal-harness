# PDF·OCR 도구와 환경 제약 (현행)

PDF 생성·텍스트 추출·OCR 교정의 현행 지식. 10단계 PDF 생성이나 자산 추출·판례 변환 작업 전 이 노드를 읽는다.

## generate_pdf의 구조: 표현과 렌더링의 분리

`generate_pdf.py`는 parse(txt에서 구조로), render(구조에서 HTML/CSS로), write_pdf(변환)의 3단이고, 표현원(CSS 모듈 상수)이 유일하다. 이 분리가 세 이득을 낳는다: 엔진 교체가 서식에 영향을 주지 않는 이식성, 한 곳 수정이 두 엔진에 반영되는 단일 수정점, 시각 검증이 막혀도 표현·텍스트 층을 검증하면 되는 검증 축 이동. 근거: [[D-0005]].

## 폴백 체인과 엔진별 보정

WeasyPrint는 GTK 런타임을 요구해 이 환경에서 실패하고, Playwright가 시스템 Edge(`channel="msedge"`)로 폴백한다(번들 다운로드 불요). Chromium은 CSS Paged Media margin box(`@bottom-center`)를 지원하지 않으므로 페이지번호는 WeasyPrint 경로에서 CSS로, Playwright 경로에서 `footer_template`으로 각각 재현한다. 폴백은 "동일 표현원 + 렌더러별 보정 레이어"의 조합이며 서식 일치는 근사이지 항등이 아니다. 근거: [[M-0006]]·[[D-0005]].

## CSS 상속 함정

`text-indent`는 상속 속성이라 `p` 규칙이 `.sub`·`.i1`·`.i2`·`.foot p`에 전파된다. 예외에는 명시적 `text-indent: 0` override가 필수이고 특정성이 이를 보장한다. 표현원의 표현력을 넓힐수록 예외 봉쇄를 명시해야 한다. 근거: [[M-0009]].

## 이 환경의 검증 제약과 갈음 수단

poppler(pdftoppm) 부재로 PDF 시각 렌더가 막히고 Playwright MCP는 file: 프로토콜을 차단한다. 갈음: `--html`로 구조 검증, PyMuPDF 텍스트 추출로 페이지·footer 검증, `--verify` 옵션(생성 직후 footer `- N -`·제목 자동 점검, PyMuPDF 미설치면 생략). 승격 정본: `generate_pdf.py --verify`([[M-0018]]), `harness/quality/PDF_서식.md`. 근거: [[M-0009]]·[[D-0005]].

## 벡터 PDF 강제 OCR (승격)

텍스트레이어가 없는 벡터 path PDF(웹캡처)는 600dpi 강제 렌더 + Tesseract psm 6(단일 블록)이 결정적이다(psm 3은 다단 레이아웃을 파편화: 실측 3,525자 대 4,846자). 판별은 get_text 길이와 get_drawings 수. drawings 수만으로는 벡터-글자 PDF와 정상 표-많은 PDF를 구별할 수 없고(경북대 학칙이 반례: textlen 21만에 drawings 29만) text 길이가 유일 지표다. 승격 정본: `harness/관리_절차.md`의 벡터 PDF 강제 OCR 소절([[M-0025]]), `extract_all.py` 벡터 분기 600dpi·psm6([[M-0026]]). 근거: [[M-0020]]·[[M-0023]].

## OCR 교정의 방법

두 독립 OCR(기존 교정본 + 신규 psm6)은 상보적이라 교차대조로 대부분의 깨진 토큰이 복원된다. 기존 교정본이 이미 양호하면 전면 재교체는 후퇴다(웹잔재 재유입). 개별 토큰만 보강한다. 둘 다 판독 불가면 추측하지 않고 `[OCR 불확실]`을 유지한다(익명화 약칭·발언 첫머리 등). 근거: [[M-0023]]·[[M-0024]].

## 이 도구 계열이 완주한 자기개선 사이클

발견(사용자 감사)에서 개선(판례 정정), 성문화(관리_절차), 코드 근본 개선(extract_all 파라미터)까지 4단계를 완주해 같은 유형 문제의 재발이 구조적으로 차단됐다. 완주가 가능했던 것은 offset이 기계적(dpi·psm 파라미터)이었기 때문이다. 판단적 offset은 코드가 아니라 판단 검증 장치로 간다([[L-0005]]의 β 삼분). 근거: [[D-0013]] 축2.
